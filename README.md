# 📰 RSS 新闻聚合器

基于 Fresh (Deno) 的现代化 RSS 新闻聚合展示应用，采用报纸式布局设计。

## ✨ 特性

- 📡 **多源 RSS 聚合** - 支持多个 RSS/Atom 源同时抓取
- 🏷️ **智能分类** - 自动将文章分类为科技、国际、商业、科学、体育等类别
- 🔖 **标签提取** - 自动提取文章关键标签
- 💾 **三级缓存** - 内存 + Deno KV + 源站的高效缓存策略
- 🔄 **后台刷新** - 定时自动刷新新闻内容
- 📱 **响应式设计** - 桌面端报纸式布局，移动端自适应
- ⚡ **边缘部署** - 完全兼容 Deno Deploy
